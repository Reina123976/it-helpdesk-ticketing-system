# IT Help Desk & Ticketing Management System

Full Stack Web Development Internship Project — Integrated Digital Systems (IDS)

## Overview

A web-based IT Help Desk & Ticketing Management System that streamlines technical
support operations. Employees submit support requests; IT support agents and
administrators manage, prioritize, assign, and resolve tickets through a
centralized dashboard.
