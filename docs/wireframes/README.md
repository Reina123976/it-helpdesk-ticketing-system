# UI Wireframes

Screens exported from Stitch, covering the core flows of the IT Ops Central
application (this internship project's working name for the Help Desk system).

| Screen | File | Notes |
|---|---|---|
| Sign In | `login.png` | Email/password + SSO option |
| Dashboard | `dashboard.png` | Open/unassigned/SLA widgets, category chart, recent activity, critical tickets |
| Ticket Queue | `ticket-list.png` | Filterable ticket list with quick stats |
| Ticket Detail | `ticket-details.png` | Ticket properties, communication thread, internal notes, audit trail |
| Reports | `reports.png` | Performance analytics: resolution time, SLA compliance, CSAT, agent leaderboard |
| Knowledge Base (Browse) | `knowledge-base.png` | Category browse, search, popular/recent articles |
| Knowledge Base (Article) | `knowledge-base-article.png` | Article detail with related articles and helpfulness rating |
| Admin / User Management | `admin-settings.png` | Profile, security, preferences, system settings tabs |

See [`../design-system.md`](../design-system.md) for the full design system
(colors, typography, spacing, component specs) these screens follow.
